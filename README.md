# FU-CRA-AUTH-GRAPHQL

Frontend for user authentication using create-react-app

## Description

Trying to apply my knowledge of Authentication and GraphQL in this simple application.
Server <https://github.com/MarrieMitsu/su-auth-graphql>

### Feature

* Login &check;
* Register &check;
* Forgot Password &check;
* Reset Password &check;
* Change Password &check;
* Update Profile &check;
* Upload Profile Picture &cross;
* Delete Account &check;

## Usage

```bash
# Install dependencies
pnpm

# Setup .env

# Development
pnpm start

# Production
pnpm build

```
